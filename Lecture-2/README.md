# Install Tensorflow

## Windows 10 - cpu
0. Download python3.6.5 - 64bit
1. Install python3.6.5(Add PATH)
2. open powershell/cmd/etc.
3. python -m pip install tensorflow
4. check install - python -m pip list
5. check list of TF
